# 💎 Crystal Collector

A fast-paced Unity 3D arcade game where you control a ball and race against
the clock to collect all the crystals scattered around the level.
Every crystal gives you a point — collect them all before time runs out!

---

## 🎮 How to Play

- Use **Arrow Keys** or **WASD** to roll the ball
- **Crystals** are placed around the level
- You have a **limited time** to collect as many as possible
- Each crystal collected gives you **+1 point**
- Try to collect **all crystals** before the timer hits zero!

---

## ✨ Features

- Smooth ball physics movement
- Crystal pickup system with score tracking
- Countdown timer with game over state
- Clean 3D level design

---

## 🛠️ Built With

- Unity 3D (2022.3 LTS or newer)
- C#

---

## 🚀 Getting Started

1. Clone the repo
   git clone https://github.com/your-username/crystal-collector.git

2. Open Unity Hub → Add project from disk → select the cloned folder

3. Open Assets/Scenes/MainScene.unity

4. Press Play and enjoy!
